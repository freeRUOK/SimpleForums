# SimpleForums
一个简易盲人论坛聚合Android客户端

## 功能说明

  目前包括如下网站：

- [爱盲论坛](https://www.aimang.net/)
- [帮忙社区](http://bbs.abm365.cn/)
- [蜻蜓社区](http://www.qt.hk/)
- [争渡网](http://www.zd.hk/)

  上述网站可以使用的功能包括：

* 分类浏览和搜索浏览
* 用户登录
* 用户发帖和回帖
* 本地收藏和导出压缩包
* 播放关联媒体
* 语音朗读内容

待完成或者完善的功能包括：

* 文章排版
* 关联资源下载

## 安全和隐私说明

  在用户设备本地存储了如下数据， 其他一律提交关联网站：

* 网站cookie | 用于用户登录发帖和回帖
* 用户名和授权码 | 用于关联网站用户身份识别
* 用户偏好配置和未发布的草稿 | 方便用户使用程序
